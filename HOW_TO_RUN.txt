ALTERNATIVE METHODS:
-------------------

METHOD 1: PowerShell
--------------------
Right-click "start.ps1" → Run with PowerShell

METHOD 2: VS Code
-----------------
1. Open this folder in VS Code
2. Press F5
3. Select "Launch Chrome - Merkle AI Platform"

METHOD 3: Command Line
----------------------
Open terminal in this folder and run:
    npm install
    npm run dev

METHOD 4: VS Code Terminal
--------------------------
1. Open VS Code
2. Press Ctrl+` to open terminal
3. Run: npm install
4. Run: npm run dev

TROUBLESHOOTING:
---------------

Problem: "node is not recognized"
Solution: Install Node.js from https://nodejs.org/

Problem: Port already in use
Solution: Close any other development servers and try again

Problem: Page looks broken
Solution: Press Ctrl+Shift+R to hard refresh

Problem: npm install fails
Solution: Check your internet connection, try again
